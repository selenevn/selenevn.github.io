    Đưa bài lên web thế nào?

    Trang web vẫn chưa hoàn thiện. Tao tạm dùng github để làm cái blog ghi chép lại những ý tưởng vặt vãnh nảy sinh trong quá trình hoàn thiện.

    Với ý tưởng làm trang web giới thiệu các mặt hàng handmade (chủ yếu là đồ da), thì việc lập và vận hành một trang web là việc thừa thãi. Tao không có nhiều thông tin để post lên thường xuyên, và cũng không muốn mất nhiều thì giờ để học quản lý trang web. Ý tưởng của tao là chỉ mua tên miền rồi gắn cái tên miền ấy vào một repo trên github.

    Chỉ là một trang web tĩnh, đơn giản vậy thôi.

    Đây là các bước để update thông tin

    1. Clone repo đã lập cho trang web (https://github.com/selenevn/selenevn.github.io) lưu tại D:/www/selenevn.github.io/.
    2. Dùng wamp tạo virtualhost cho thư mục trên, với tên miền selene.dev (chỉ sử dụng ở máy tính).
    3. Dùng publii (https://getpublii.com/) để làm trang web tĩnh. Thông tin sau khi update có thể kiểm tra thông qua preview của phần mềm hoặc copy toàn bộ các file sinh ra trong thư mục output sang virtualhost để xem trên trình duyệt.
    4. Khi ok toàn bộ rồi thì chỉ cần commit và push lên github (từ virtualhost) là xong.
